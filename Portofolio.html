Futsal

Foto di samping diambil ketika saya mengikuti turnamen futsal bersama tim saya. Saya berposisi sebagai Anchor tetapi terkadang saya berpindah posisi menjadi Pivot ketika pertandingan berlangsung.

Saya juga memiliki pemain favorit yaitu Evan Soumilena pemain dari Black Steel Manokwari dan sekaligus pemain Timnas Indonesia berposisi sebagai pivot dan saya memfavoritkan dia karena cara bermain dia yang keras dan ngotot untuk mencetak gol.

E-Sport

Saya juga berpartisipasi dalam dunia E-Sport. Untuk saat ini saya masih aktif dalam E-Sport terutama untuk FIFA. Pencapaian saya dalam dunia E-Sport menjadi juara 2 dan juara 3 dalam turnamen FIFA yang diadakan oleh salah satu Universitas di Bandung.

Selain FIFA saya juga pernah menjadi Player dan Coach dalam permainan Arena Of Valor, tetapi untuk Arena Of Valor sendiri saat ini saya sudah tidak aktif di permainan tersebut.

Gym

Saya saat ini juga mendalami dunia angkat beban, saya sudah berlatih dari tahun 2017 jadi sudah sekitar 5 tahun saya berada di dunia gym. Bukan hal yang mudah untuk menjaga konsisten dalam berlatih angkat beban, selain itu juga terkadang badai cedera menyerang saya karena saya terlalu mendorong diri saya ketika berlatih.
